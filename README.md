# JavaScript Console Quiz – Lab Assignment 4

This is a simple JavaScript  quiz game that runs inside the browser console.  
The quiz asks multiple questions using `prompt()` and gives feedback with `alert()`.  
At the end, it shows the final score.

## How to Run
1. Open any browser (Chrome recommended)
2. Right-click → Inspect → Console
3. Paste the entire  code from `quiz.js`
4. Press  Enter and answer the prompts

## Features
- Uses arrays, loops, functions, and conditionals
- Normalizes input with `toLowerCase()` and `trim()`
- Gives immediate feedback after each question
- Tracks and displays final score
- Contains 5 questions as required

## Files Included
- `quiz.js` — main  quiz program

## Breakpoints Tested
- Works on all  modern browsers (Chrome, Edge, Firefox)
